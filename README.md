# FSM
利用有限狀態機的方法模擬出車站的自動售票販賣機

![說明](https://user-images.githubusercontent.com/95120819/192145519-81ea758f-3969-4b11-a5fd-3796a7dd615a.png)

可以投入零錢種類為1元、5元、10元、50元
 ( 不可同時投入多個硬幣，
       例如: 一次投入2個1元 )
票數的選擇為 1～5張 
車站為 S1、S2 、 S3 、 S4 、 S5 共五站
選站時必須選擇 起點站 與 終點站
當起站與終站相同時 則為 月台票
